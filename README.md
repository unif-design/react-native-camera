# @unif/react-native-camera

基于 react-native-vision-camera 的相机库（单拍/连拍/视频+水印+预览）

## Installation


```sh
npm install @unif/react-native-camera
```


## Usage


```js
import { multiply } from '@unif/react-native-camera';

// ...

const result = multiply(3, 7);
```
